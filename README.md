# Plantas_en_juego
Juego de cartas de **biotecnología vegetal** para estudiantes de liceo.  
Cada grupo recibe una planta y, usando cartas de **genes**, **herramientas biotecnológicas** (CRISPR, RNAi, edición epigenética, cultivo in vitro, etc.) y **eventos de estrés** (sequía, salinidad, patógenos, UV, frío), debe diseñar la mejor estrategia para que su cultivo **sobreviva y cumpla la misión productiva**.  
Pensado para usar en clase de **Biología/Biotecnología**, trabajando conceptos de genética, estrés en plantas y toma de decisiones científicas de forma lúdica.
